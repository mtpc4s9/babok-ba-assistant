# Top 20 BABOK V3 Techniques Index

This folder contains deep-dive execution guidelines and templates for the **Top 20 Business Analysis Techniques** as defined by the IIBA® BABOK® Guide v3.

## Overview
These 20 techniques represent the primary practical methods utilized by Business Analysts to plan, elicit, specify, model, verify, validate, and evaluate requirements across all enterprise project types.

---

## 📂 Navigation & Folder Structure

Most techniques are stored as flat Markdown files (`.md`) directly in this directory for rapid lookup and low token consumption. However, **three high-complexity modelling techniques** are structured inside dedicated subdirectories:
1.  **10.35 Process Modelling:** Located at `10.35-process-modelling/`
2.  **10.47 Use Cases and Scenarios:** Located at `10.47-use-cases-and-scenarios/`
3.  **10.48 User Stories:** Located at `10.48-user-stories/`

## 🤖 AI Agent Execution Instructions
1.  **Selection Rule:** When a task refers to a specific technique ID, immediately load the corresponding file from this folder.
2.  **Context Loading:** For Process Modelling, Use Cases, and User Stories, navigate into their subfolders to load specific templates or scripts.
3.  **Anti-Hallucination:** Strictly adhere to the parameters and structures outlined in each technique file. Do not invent custom columns or attributes that violate the mapped templates.
